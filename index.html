One-time setup

This board needs a free Supabase project as its shared database.

  1. Go to supabase.com → create a free account and a new project.
  2. In the project: SQL Editor → paste and run the SQL from README.md.
  3. Settings → API: copy the Project URL and anon public key.
  4. Open index.html and paste both into the CONFIG block at the bottom.
  5. Commit and push. Done — everyone with the link shares the same board.
